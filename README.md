## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/AmoiiKane/dev-portfolio.git
cd dev-portfolio

# Install dependencies
npm install

# Run the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Featured Projects

### [`sysmon-cli`](https://github.com/AmoiiKane/sysmon-cli)
Real-time system resource monitor for the terminal — CPU, RAM, disk, network, and top processes with live refresh.  
`Python` `psutil` `rich`

### [`automate-me`](https://github.com/AmoiiKane/automate-me)
File organizer bot that watches a directory and auto-sorts files into categorized subfolders by type.  
`Python` `watchdog` `rich`

### [`c-chat`](https://github.com/AmoiiKane/c-chat)
Terminal chat app in C using TCP sockets and POSIX threads — supports 32 concurrent clients.  
`C` `pthreads` `BSD Sockets`
